Privacy Policy

DB Grapher web server keeps track of IP addresses users use to access the application. This information is used to moderate the website and prevent abuse.

DB Grapher has third party Google drive integration. By user's permission, it accesses the following data

  • Email: is used to indicate which Google account is in use.
  • Account avatar: is used to further emphasize the Google account in use.
  • Drive file read: is used to open files.
  • Drive file write: is used to save files.
