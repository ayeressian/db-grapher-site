Privacy Policy

DB Grapher web server keeps track of IP addresses users use to access the application. This information is used to moderate the website and prevent abuse.

DB Grapher has third party Google Drive integration. By user's permission, it accesses the following data

  • Email, user name, avatar: is used to indicate which Google account is in use.
  • Drive file read permission: is used to open DB Grapher files.
  • Drive file write permission: is used to save DB Grapher files.
Users can opt out of using Google Drive integration and use their local computer for storing files. We don't collect or distribute any user information.
